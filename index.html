FigmaLand

Projeto desenvolvido como uma landing page moderna criada com React, Vite e TailwindCSS para aprimorar habilidades em front-end e explorar o uso de Tailwind pela primeira vez. O projeto foi inicializado com React e Vite, aproveitando o suporte ao JSX para uma estrutura de componentes eficiente e responsiva. FigmaLand é um exemplo de design de interface limpa e focada em experiência de usuário, com uma estilização personalizada para adaptar a interface a diferentes tamanhos de tela.

Ideal para portfólio ou páginas de captura de leads, Figmaland oferece uma base sólida para desenvolvimento de landing pages versáteis e escaláveis.

Cotações ao vivo

Este projeto tem como objetivo fornecer cotações em tempo real diretamente da B3 (Bolsa de Valores do Brasil) usando uma integração com o MetaTrader. O código busca, conecta e atualiza dados de mercado, tornando-o ideal para investidores, analistas e qualquer pessoa que precise monitorar os preços de ativos ao vivo.

Clique em Google Colab para saber sobre: Importação de Bibliotecas, Inicialização do MetaTrader, Seleção de Símbolos Disponíveis e Obtenção de Cotações ao Vivo.

Costs

Costs é um gerenciador de projetos desenvolvido em React, focado na organização e no controle financeiro dos projetos. Com o Costs, o usuário pode criar novos projetos, definir um orçamento para cada um deles e categorizá-los de acordo com sua finalidade, facilitando a visualização e o gerenciamento de múltiplos projetos ao mesmo tempo.

Este sistema é ideal para freelancers, pequenas empresas e qualquer pessoa que precise manter seus projetos organizados e em conformidade com o orçamento planejado, tudo em uma interface intuitiva e fácil de usar.
